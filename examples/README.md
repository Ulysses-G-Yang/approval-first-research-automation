# Reproducible Examples

These examples use self-authored local data. They are designed to show the task workspace and approval model without requiring a provider key or third-party website.

| Example | Input | Workflow | Result |
| --- | --- | --- | --- |
| [Research report](research-report/README.md) | CSV | `file_report` | Source-backed local Markdown report. |
| [Document to Markdown](document-to-markdown/README.md) | Markdown plus SVG asset | `document_to_markdown` | Markdown bundle with a task-local asset. |
| [Offline draft package](content-draft/README.md) | Markdown plus SVG asset | `content_save_draft` | Juejin, Zhihu, or CSDN package with `published: false`. |

For selector drift and extraction recovery, use the separate [Page Evolution Lab](../labs/page_evolution/README.md).
