# Educational Archive: v1.0-educational

`v1.0-educational` preserves the project's earlier single-site implementation as a historical teaching reference. It is not the active development line and should not be treated as a production-ready collector.

The current `feat/general-crawler` branch separates reusable execution behaviour from site configuration. The Taobao configuration remains a typical educational case for studying configuration migration, structured data extraction and failure handling.

Use the archive only in lawful, permissioned environments. Do not use it to bypass access controls, collect non-public data or violate a service's terms.
