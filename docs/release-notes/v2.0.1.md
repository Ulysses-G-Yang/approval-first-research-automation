# v2.0.1 - 通用框架首次可验证发布

## Highlights

- 配置驱动的 Playwright 执行引擎，支持字段、分页、动作和浏览器配置。
- 三层提取路径：常规 selector、Scrapling 自适应解析、可选 LLM selector 修复。
- 豆瓣正常与故障注入验证，关键字段断言和离线单元测试。
- 安全的本地浏览器状态管理，避免 Cookie 与 API 密钥进入 Git。
- 完整的 README、教育归档说明、贡献与安全政策、Issue 模板和离线 CI。

## Upgrade note

`v2.0.0` 的 Release 文案早于源码同步发布；该标签保持不变，实际可验证源码请使用本版本。

## Quick start

```bash
conda env create -f environment.yml
conda activate generic-crawler-py312
playwright install chromium
python extract_prices.py --config configs/douban.yaml --output output/douban.jsonl
```
