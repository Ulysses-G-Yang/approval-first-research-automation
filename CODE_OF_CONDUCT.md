# Code of Conduct

We are committed to a respectful, constructive and inclusive community.

Be kind, assume good intent, give actionable technical feedback and avoid harassment, discrimination or personal attacks. Do not post credentials, private data, copyrighted page captures beyond fair discussion, or instructions intended to evade access controls.

Report concerns privately through the repository security contact described in `SECURITY.md`. Maintainers may edit, remove or close content that violates these expectations.
